# AI-Assisted Microcontroller Selection for IoT

## Overview

This project automates microcontroller (MCU) selection for embedded IoT
applications using Machine Learning. It reduces manual effort, improves
selection accuracy, and validates configurations through virtual
simulation --- eliminating early dependence on physical hardware.

## Key Highlights

-   Automated MCU selection using Random Forest
-   Synthetic dataset with thousands of configurations
-   JSON-based hardware configuration export
-   Automatic Renode (.repl) platform file generation
-   Functional & hardware-level simulation validation

## Tech Stack

Python • Scikit-learn • JSON • Renode • Google Colab

## Impact

-   Faster embedded system design cycle
-   Reduced hardware prototyping cost
-   Improved accuracy in MCU suitability selection
-   Fully automated end-to-end pipeline

## Future Improvements
